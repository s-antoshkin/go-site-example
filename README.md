# Golang site example
Пример представляет собой небольшой сайт - "приглашение на вечеринку": формы, макеты, взаимодействие с БД `Postgres(pgx)` + тесты.

## Requirements
Go 1.19 or above.

## Installation
```
git clone https://github.com/s-antoshkin/go-site-example
cd go-site-example
go get -u
```

## Start the application
```
go run .
```
